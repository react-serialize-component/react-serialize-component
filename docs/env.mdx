# 环境设置

## 主要设置功能
1. 可以通过环境设置设置ajax接口请求相关的 参数和返回值逻辑
2. 可以通过env设置设置 alert，toast， confirm的默认方法

``` js
import Core from '@react-serialize-component/@react-serialize-component/antd'
Core.setEnv({
  proxy: {
    // 拦截所有request做出处理
    request: {
      '.*': function all(result) {
        // console.log(result);
        return result;
      },
    },
    // 拦截所有 response请求做出处理
    response: {
      '.*': function all(result) {
        return result;
      }
    },
  },
  // 设置alert方法
  alert(msg) {
    // eslint-disable-next-line no-alert
    window.alert(msg);
  },
  // 设置confirm方法
  confirm(msg, title) {
    // eslint-disable-next-line no-alert
    // eslint-disable-next-line no-restricted-globals
    const res = window.confirm(msg);
    return Promise.resolve(res);
  },
  // 设置toast方法
  toast(msg, time) {
    toast('aaa');
  }
});
```
